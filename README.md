# Zombie Land - Pygame Project

- Style: _2D Top Down Shooter_
- Theme: _Zombie Apocalypse_

## Features

- Menu System
- Available many characters
- Enemy AI with path-finding ability
- Collectable Items
- Level created with **Tiled**
